## Laravel Graphql Phonebook APP Test

```bash
# Clone via ssh
$ git clone git@github.com:BryanJecie/phonebook.git

# Install dependencies
$ composer install

# Config database
$ cp .env.example .env

# Run commands
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed

# Run app via local
$ php artisan serve

# Credentials
$ Email: test@example.com
$ Password: secret
```
